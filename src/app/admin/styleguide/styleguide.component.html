Icon Styles

Outline Style

Clean line icons with consistent 1.5px stroke width

Primary icon style for interface elements

Solid Style

Filled icons with solid color and simple shapes

Used for smaller UI elements and navigation

Duotone Style

Two-tone icons with accent and background colors

Used for feature highlights and visual emphasis

Icon Usage Rules

  • • Maintain consistent size and style within the same context
  • • Use outline style for most interface elements and navigation
  • • Use solid style for smaller icons (16px) or when emphasis is needed
  • • Reserve duotone style for key feature highlights or call-to-actions
  • • Maintain adequate spacing around icons (at least 8px on all sides)
  • • Use primary brand color for interactive icons, neutral colors for decorative ones

Illustration Styles

Outline Style

Minimalist, clean outline illustrations with 1px stroke.

Used for empty states and simple concepts

Flat Style with Depth

Flat color illustrations with subtle shadows for depth.

Used for featured content and onboarding

Isometric Style

3D-like illustrations with isometric perspective.

Used for complex concepts and data visualization

Images & Photography

Image Style Guidelines

  • • Use high-quality, high-resolution images (min. 1200px wide)
  • • Prefer natural, authentic photography over obvious stock photos
  • • Maintain consistent color treatment across all images
  • • Use images that reflect diversity and inclusivity
  • • Crop images to maintain focal point and subject visibility
  • • Apply consistent aspect ratios within content sections

Image Formats

• JPG: For photographs and complex images

• PNG: For graphics with transparency

• WebP: Preferred for web when supported

• SVG: For illustrations and simple graphics
